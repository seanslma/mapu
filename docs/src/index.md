## \(\mathrm{\LaTeX}\) math example
The standard form of a quadratic equation is:
\[ax^2 + bx + c = 0\]

The solution to this equation is given by the quadratic formula:
\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]
